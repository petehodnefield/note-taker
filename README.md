# Monga Note Taker

## Description

- Monga Note Taker is a simple notetaking app. A user can enter the title of the note, and the note contents. Once both sections are entered, a save icon appears that allows the user to save the notes.

## Installation

Requirements:

- node js
- express
- heroku
